# CodeSnippetsSync

> 这是一款帮助你在多台Mac设备中同步Xcode CodeSnippets的App，它利用CloudKit技术帮你将CodeSnippets自动同步到你的iCloud私人数据库

## 特点

- 所有数据在iCloud的私人数据库存储，除了你自己任何人无法访问这些数据

- 使用AppKit原生开发，空间占用少


## 更新日志

### [1.1] - 2023-10-04

- 现在 CodeSnippetsSync 可以监听Xcode新增，编辑，删除的操作，立即同步到iCloud
